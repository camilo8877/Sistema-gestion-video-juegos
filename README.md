Sistema de Gestión de Tienda de Videojuegos

Este proyecto es un sistema de consola desarrollado en Python para administrar una tienda de videojuegos.  
Permite gestionar inventario, registrar ventas, consultar estadísticas y realizar búsquedas de videojuegos.

---

#  Características

El sistema permite:

✅ Agregar videojuegos  
✅ Mostrar inventario disponible  
✅ Buscar videojuegos por código  
✅ Actualizar precios  
✅ Registrar ventas  
✅ Mostrar estadísticas generales  
✅ Eliminar videojuegos  
✅ Buscar videojuegos por plataforma  
✅ Mostrar videojuegos con bajo stock  
✅ Mostrar historial de ventas  
✅ Mostrar el videojuego más vendido
